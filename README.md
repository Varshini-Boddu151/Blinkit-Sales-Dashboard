# Blinkit Sales Dashboard (Power BI)

## 📊 Project Overview
This project analyzes Blinkit grocery sales data using Power BI to uncover insights and improve business decisions.

## 🎯 Objectives
- Analyze sales performance
- Identify top-performing categories
- Understand customer purchasing behavior

## 📌 Key Insights
- Personal Care category generated highest sales
- Snacks & Confectionery are high-demand items
- Visibility impacts product sales significantly

## 🛠 Tools Used
- Power BI
- SQL
- Excel


## 🚀 Conclusion
The dashboard helps stakeholders make data-driven decisions and improve store performance.
